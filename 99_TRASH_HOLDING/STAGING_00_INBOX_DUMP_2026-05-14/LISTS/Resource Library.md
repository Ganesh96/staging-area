# Resource Library

A curated collection of valuable links, tools, and platforms to support your journey in [[2. Learning Roadmaps]] and [[5. Interview Dojo]].

### Websites & Roadmaps
- **Roadmap.sh**: Visual roadmaps for developers ([Software Architect](https://roadmap.sh/software-architect), [Backend](https://roadmap.sh/backend)).
- **Tech Interview Handbook**: Comprehensive guide for technical interviews.
- **Coding Interview University**: A full computer science study plan.
- **QuickRef.me**: Quick reference for various programming languages and tools.

### Coding & Practice
- **LeetCode**: Primary platform for practicing coding problems.
- **NeetCode.io**: Categorized problems and video solutions.
- **interviewing.io Leetcode Patterns Guide**: A guide to common LeetCode patterns.
- **HackerRank**: Coding challenges and competitions.
- **TopCoder**: For competitive-level algorithm practice.
- **visualgo.net**: Visualizations of data structures and algorithms.

### Recommended Books
- **Cracking the Coding Interview** by Gayle Laakmann McDowell.
- **System Design Interview** by Alex Xu.
- **Elements of Programming Interviews** by Adnan Aziz, Tsung-Hsien Lee, Amit Prakash.

### Job Search & Compensation
- **Simplify Jobs**: Autofill applications.
- **Levels.fyi**: Salary comparison tool for tech jobs.
- **Layoffs.fyi**: Tracker for tech company layoffs.
