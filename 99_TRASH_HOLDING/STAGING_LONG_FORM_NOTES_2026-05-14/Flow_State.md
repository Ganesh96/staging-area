# Flow State

Flow is an optimal state of consciousness where performance and satisfaction peak.

## Flow Triggers
- Clear goals
- Challenge-skill balance
- Immediate feedback
- Deep focus

## Flow Cycle
1. Struggle
2. Release
3. Flow
4. Recovery

## Entry Rules
- No multitasking
- One task, one session
- Start before you feel ready

## Recovery
- Flow requires oscillation
- Recovery is not laziness
- Boring breaks increase flow return

## Rule
You cannot shortcut flow.
You must earn it through depth.
