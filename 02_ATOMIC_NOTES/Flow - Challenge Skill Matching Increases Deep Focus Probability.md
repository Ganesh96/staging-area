# Flow - Challenge Skill Matching Increases Deep Focus Probability

## Core idea
Flow is most likely when task difficulty is slightly above current ability with clear goals and immediate feedback.

## Explanation
Flow guidance in the sources repeatedly emphasizes calibration and structured challenge.

## Use case
Use when work is either too easy (boredom) or too hard (anxiety/stall).
