# Focus - Reactive Mornings Destroy Deep Work Availability

## Core idea
Once mornings become reactive, deep-work capacity is usually lost for the rest of the day.

## Explanation
The notes consistently prioritize deep work first and separate shallow tasks later; this implies morning protection is a leverage point.

## Use case
Use as a hard rule when scheduling daily blocks.
