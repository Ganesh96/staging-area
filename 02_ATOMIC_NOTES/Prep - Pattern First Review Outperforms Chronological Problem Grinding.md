# Prep - Pattern First Review Outperforms Chronological Problem Grinding

## Core idea
Company interview prep is more effective when problems are grouped by patterns and failure modes, not solved in raw list order.

## Explanation
The Doordash sources repeatedly organize by blocks (graphs, trees, design, heaps), indicating pattern clustering is the operational unit.

## Use case
Plan weekly sessions by pattern clusters and revisit weak clusters with timed sets.
