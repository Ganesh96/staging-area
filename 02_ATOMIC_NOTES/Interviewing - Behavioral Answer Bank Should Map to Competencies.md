# Interviewing - Behavioral Answer Bank Should Map to Competencies

## Core idea
A compact answer bank should be indexed by competency (leadership, conflict, pressure, failure, communication), not by random prompts.

## Explanation
Many questions are variations of the same competency test. Mapping stories to competencies reduces prep load and improves consistency.

## Use case
Build a one-page prep sheet before interviews.
