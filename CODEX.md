# CODEX Instructions

This repo is a staging system for organizing raw personal knowledge, notes, references, projects, and journals.

Primary objective:
Transform messy input into structured, retrievable, deduplicated knowledge without losing original meaning.

Rules:
- Never delete content permanently.
- Move uncertain deletions to `99_TRASH_HOLDING/`.
- Preserve emotional/journal writings unless explicitly instructed.
- Do not rewrite personal content unless asked.
- Prefer classification, cleanup, deduplication, and metadata extraction.
- Keep raw source material traceable.
- When unsure, ask before restructuring.

Workflow:
1. Raw input goes into `00_INBOX_DUMP/`.
2. Active cleanup happens in `01_STAGING/`.
3. Clean, organized outputs go to `02_READY_FOR_NOTION/`.
4. Trash candidates go to `99_TRASH_HOLDING/`.

Classification targets:
- Journal / emotional writing
- Knowledge base
- Projects / tasks / plans
- References / links / PDFs
- Unknown / needs review
