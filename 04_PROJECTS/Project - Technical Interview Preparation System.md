# Project - Technical Interview Preparation System

## Protocol tracks
1. DSA track (patterns, timed practice, review).
2. System design track (component tradeoffs, whiteboard communication, case walkthroughs).
3. Behavioral track (STAR/PAR stories + mock loops).

## Weekly cadence
- 3 DSA sessions
- 2 system design sessions
- 2 behavioral mock sessions
- 1 weekly retrospective

## Scorecard
- [ ] DSA sessions completed
- [ ] System design sessions completed
- [ ] Behavioral mocks completed
- [ ] Story bank updated with feedback

## Risks / blockers
- Checklist completion without deliberate practice depth.
- Over-expansion of resources without rehearsal cycles.
