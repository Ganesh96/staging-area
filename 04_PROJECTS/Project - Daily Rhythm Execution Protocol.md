# Project - Daily Rhythm Execution Protocol

## Daily protocol
1. Morning anchor (Move → Reflect → Grow).
2. Protect first work block for deep work.
3. Batch shallow work into a separate block.
4. End day with shutdown ritual.
5. Enforce digital sunset one hour before sleep.

## Weekly scorecard
- [ ] Morning anchor completed 5+ days.
- [ ] Deep work block completed 5+ days.
- [ ] Shutdown ritual completed 5+ days.
- [ ] Digital sunset completed 5+ days.

## Metrics
- Deep work hours/week.
- Reactive-morning count/week.
- Sleep start-time consistency (days within target window).

## Risks / blockers
- Phone-first mornings.
- Evening spillover and late-night screen drift.
