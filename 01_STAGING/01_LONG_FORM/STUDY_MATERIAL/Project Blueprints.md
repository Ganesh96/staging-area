# Project Blueprints

Ideas and architecture summaries for projects of varying complexity. Use these as a sandbox to apply concepts from [[3. Technical Deep Dives]].

### Small to Medium Projects
- **[[Project Kangiten No Sho]]**: A web application using React/RedwoodJS, GraphQL, SQL (PostgreSQL), and Redis. This is a great project for mastering the full stack.

### Master-Level Projects
- **[[Project Ghost Tower]]**: A document and information management platform for companies, featuring secure document storage (AWS S3), logistics/inventory management, and hierarchical user account management. This project is ideal for practicing [[Software Architecture and Design]] principles.
- **Machine Learning Projects** (from [[summer blueprint]]):
    - **Image Segmentation/Classification**: Identify an area over a road or map using deep learning (PyTorch).
    - **NLP + Patch-based Project**: A project leveraging natural language processing.
