# 12-Week Study Plan and Leetcode Map

Purpose
This plan organizes the extracted Leetcode and technical-prep material into a practical study sequence. The uploaded PDF titled "12 Week Programme - Leetcode" includes a broad topic list; this file turns it into an executable preparation plan.

Study allocation model
For most SDE interviews, prioritize:
- 80-90% coding/data structures/algorithms if coding is the main weakness
- system design and behavioral practice in parallel
- weekly review of OS/networking/databases
- daily verbal explanation practice

Weekly structure
Each week:
- 5 coding practice days
- 1 design/behavioral practice day
- 1 review/mock day

Daily coding block:
1. Review concept: 20-30 minutes
2. Solve 2-4 problems: 90-150 minutes
3. Write notes: 15 minutes
4. Re-solve one previous problem without looking: 30 minutes
5. State complexity aloud for each solution
