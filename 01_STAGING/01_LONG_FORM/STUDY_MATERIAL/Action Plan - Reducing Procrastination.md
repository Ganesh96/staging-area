# Action Plan - Reducing Procrastination

## Goal

Minimize procrastination during focused work periods.

---

## Steps

- Use 2-Minute Rule for small tasks.
- Break large tasks into smaller sub-tasks.
- Use physical triggers to start (e.g. Post-it note with task).
- Remove distractions from workspace.
- Schedule “boring breaks” between work sessions.

---

## Metrics

- % of work sessions started without delay.
- Number of completed Pomodoros per day.
