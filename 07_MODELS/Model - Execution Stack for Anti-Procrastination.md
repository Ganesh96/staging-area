# Model - Execution Stack for Anti-Procrastination

## Core model
Combine three layers: prioritization (Eisenhower), scheduling (Time Blocking), and focus cadence (Pomodoro).

## Mechanism
Choose high-priority work first, assign a block, then execute in short focus intervals with breaks.

## When to use
- Days with many competing tasks and low activation energy.

## When not to use
- Crisis tasks that require uninterrupted long sessions.

## Signals / diagnostics
- If task switching is high, batching and stricter block boundaries are missing.
