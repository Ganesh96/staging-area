# Model - Habit and Flow Execution Ladder

## Core model
Progress from intention to high-output flow in five steps: micro-start → streak stabilization → time protection → flow trigger design → recovery.

## Mechanism
- Micro-start (2-minute / 60-second rule) reduces activation energy.
- Streak systems and habit stacking increase consistency.
- Time blocking and batching protect focus windows.
- Flow triggers (clear goals, challenge-skill match, reduced distraction) increase deep immersion.
- Recovery prevents collapse and preserves repeatability.

## When to use
- Inconsistent task starts, procrastination spikes, and stop-start productivity patterns.

## When not to use
- Immediate crisis response work where rigid structure is less useful.

## Signals / diagnostics
- If starts are delayed daily, step 1 is failing.
- If starts happen but consistency breaks, step 2 is failing.
- If consistency exists but deep output is low, steps 3-4 are failing.
