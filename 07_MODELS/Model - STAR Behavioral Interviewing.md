# Model - STAR Behavioral Interviewing

## Core model
Structure behavioral responses as Situation, Task, Action, Result.

## Mechanism
For each question, keep context short, make Action specific, and quantify Result where possible.

## When to use
- Conflict, leadership, failure, deadline, stakeholder, and adaptability questions.

## When not to use
- Purely factual yes/no or highly technical deep-dive questions.

## Signals / diagnostics
- Weak answer signal: long background, vague action, no measurable result.
