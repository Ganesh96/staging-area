# Staging Protocol

For each inbox item:
1. Identify content type.
2. Preserve original meaning.
3. Extract useful ideas, tasks, references, or decisions.
4. Deduplicate repeated content.
5. Move cleaned output into the correct staging folder.
6. If ready for export, move to `02_READY_FOR_NOTION/`.
7. If unclear, mark as `NEEDS_REVIEW`.
